---
title: "Organization object"
---

This object represents an organization.  

## Schema[](#schema "Direct link to heading")

```
{
  "organization": {
    "name": "Name1",
    "created_at": "2022-05-02T13:04:09Z",
    "webhook_url": "https://test-example.example",
    "country": "CZ",
    "address_line1": "address1",
    "address_line2": null,
    "state": "state1",
    "zipcode": "10000",
    "email": "org@email.com",
    "city": "city125",
    "legal_name": null,
    "legal_number": null,
    "timezone": "Europe/Paris",
    "billing_configuration": {
      "invoice_footer": "footer custom",
      "invoice_grace_period": 3,
      "vat_rate": 15.0
    }
  }
}

```

| Attributes                                                       | Description                                                                |
| ---------------------------------------------------------------- | -------------------------------------------------------------------------- |
| **name**   String   Not null                                     | Name of the organization                                                   |
| **created\_at**   String   Not null   _ISO 8601 datetime in UTC_ | Creation date                                                              |
| **webhook\_url**   String                                        | Webhook URL                                                                |
| **country**   String_ISO 3166 (alpha-2)_                         | Country where the organization is located                                  |
| **address\_line1**   String                                      | Address (first line)                                                       |
| **address\_line2**   String                                      | Address (second line)                                                      |
| **state**   String                                               | State where the organization is located                                    |
| **zipcode**   String                                             | Zip code associated with the organization's address                        |
| **email**   String                                               | Business email address                                                     |
| **city**   String                                                | City where the organization is located                                     |
| **legal\_name**   String                                         | Legal name of the organization                                             |
| **legal\_number**   String                                       | Legal number of the organization                                           |
| **timezone**   String_TZ database_                               | Timezone of the organization - [learn more](/docs/api/resources/timezones) |

## Billing Configuration Object[](#billing-configuration-object "Direct link to heading")

| Attributes                       | Description            |
| -------------------------------- | ---------------------- |
| invoice\_footer   String         | Custom invoice footer  |
| invoice\_grace\_period   Integer | Grace period (in days) |
| vat\_rate   Float                | Custom VAT rate        |
