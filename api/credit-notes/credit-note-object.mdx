title: "Credit note object"
---

This object represents a credit note emited to re-credit a customer for a fee or an invoice.

| Attributes                                                         | Description                                                                                                                                                                                                                                        |
| ------------------------------------------------------------------ | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **lago\_id**   String   Not null                                   | Unique identifer of the credit note in Lago application                                                                                                                                                                                            |
| **sequential\_id**   Integer   Not null                            | Credit note sequential id scoped on the invoice                                                                                                                                                                                                    |
| **number**   String   Not null                                     | Credit note unique number                                                                                                                                                                                                                          |
| **lago\_id**   String   Not null                                   | Unique identifer of the invoice in Lago application                                                                                                                                                                                                |
| **invoice\_number**   String   Not null                            | Invoice unique number                                                                                                                                                                                                                              |
| **issuing\_date**   String   Not null_ISO 8601 date_               | Date of creation of the credit note                                                                                                                                                                                                                |
| **reason**   String   Not null                                     | Issuing reason of the credit note **Possible values**\-duplicated\_charge\-product\_unsatisfactory\-order\_change\-order\_cancellation\-fraudulent\_charge\-other                                                                                  |
| **credit\_status**   String                                        | Status of the credit part of the credit note **Possible values**\- available: An amount remain available for future usage.\- consumed: The amount is fully consumed. Remaining amount is 0\- voided: The remaining amount cannot be used any more. |
| **refund\_status**   String                                        | Status of the refund part of the credit note **Possible values**\- pending: The refund is pending for its execution.\- succeeded: The refund has been executed\- failed: The refund failed to execute                                              |
| **total\_amount\_cents**   Integer   Not null                      | Total amount in cents                                                                                                                                                                                                                              |
| **total\_amount\_currency**   String  Not null                     | Currency of the total amount                                                                                                                                                                                                                       |
| **credit\_amount\_cents**   Integer   Not null                     | Credit amount in cents                                                                                                                                                                                                                             |
| **credit\_amount\_currency**   String  Not null                    | Currency of the credit amount                                                                                                                                                                                                                      |
| **refund\_amount\_cents**   Integer   Not null                     | Refund amount in cents                                                                                                                                                                                                                             |
| **refund\_amount\_currency**   String  Not null                    | Currency of the refund amount                                                                                                                                                                                                                      |
| **balance\_amount\_cents**   Integer   Not null                    | Remaining credit amount in cents                                                                                                                                                                                                                   |
| **balance\_amount\_currency**   String  Not null                   | Currency of the balance amount                                                                                                                                                                                                                     |
| **vat\_amount\_cents**   Integer   Not null                        | VAT amount in cents                                                                                                                                                                                                                                |
| **vat\_amount\_currency**   String  Not null                       | Currency of the VAT amount                                                                                                                                                                                                                         |
| **sub\_total\_vat\_excluded\_amount\_cents**   Integer   Not null  | Subtotal excluding tax, in cents                                                                                                                                                                                                                   |
| **sub\_total\_vat\_excluded\_amount\_currency**   String  Not null | Currency of the subtotal amount                                                                                                                                                                                                                    |
| **created\_at**   String  Not null_ISO 8601 datetime_              | Date and time of creation of the credit note in UTC                                                                                                                                                                                                |
| **updated\_at**   String  Not null_ISO 8601 datetime_              | Date and time of last update of the credit note in UTC                                                                                                                                                                                             |
| **file\_url**   String                                             | File url of the credit note PDF                                                                                                                                                                                                                    |
| **items**   JSON                                                   | Array of [item](#credit-note-object) object                                                                                                                                                                                                        |

## Item object[](#item-object "Direct link to heading")

| Attributes                              | Description                                                  |
| --------------------------------------- | ------------------------------------------------------------ |
| **lago\_id**   String   Not null        | Unique identifer of the credit note item in Lago application |
| **amount\_cents**   Integer   Not null  | Amount in cents                                              |
| **amount\_currency**   String  Not null | Currency of the amount                                       |
| **fee**   JSON  Not null                | [Fee](/docs/api/invoices/invoice-object#fee-object) object   |
