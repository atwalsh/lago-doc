---
title: "Coupon object"
---

This object represents a coupon.  

## Schema[](#schema "Direct link to heading")

```
{
  "coupon": {
    "lago_id": "b7ab2926-1de8-4428-9bcd-779314ac129b",
    "name": "coupon_name",
    "code": "coupon_code",
    "amount_cents": 5000,
    "amount_currency": "USD",
    "percentage_rate": null,
    "coupon_type": "fixed_amount",
    "reusable": true,
    "frequency": "recurring",
    "frequency_duration": 3,
    "expiration": "time_limit",
    "expiration_at": "2022-08-08T23:59:59Z",
    "created_at": "2022-04-29T08:59:51Z"
  }
}

```

| Attributes                                                       | Description                                                             |
| ---------------------------------------------------------------- | ----------------------------------------------------------------------- |
| **lago\_id**   String   Not null                                 | Unique identifier of the coupon in Lago application.                    |
| **name**   String   Not null                                     | Coupon name.                                                            |
| **code**   String   Not null                                     | Code identifying the coupon.                                            |
| **amount\_cents**   Integer                                      | Amount in cents.                                                        |
| **amount\_currency**   String                                    | Currency of the amount.                                                 |
| **reusable**   Boolean                                           | Field specifying if coupon is reusable or not.                          |
| **expiration**   String   Not null                               | Expiration type. **Possible values**\- time\_limit\- no\_expiration     |
| **expiration\_at**   String _ISO 8601 datetime in UTC_           | Expiration date and time of the coupon.                                 |
| **coupon\_type**   String   Not null                             | Coupon type. **Possible values**\- fixed\_amount\- percentage           |
| **percentage\_rate**   Float                                     | Percentage rate of the coupon.                                          |
| **frequency**   String   Not null                                | Frequency type. **Possible values**\- once\- recurring                  |
| **frequency\_duration**   Integer                                | It determines the number of billing periods to which the coupon applies |
| **created\_at**   String   Not null   _ISO 8601 datetime in UTC_ | Date of coupon creation.                                                |

### Deprecated arguments[](#deprecated-arguments "Direct link to heading")

| Attributes                                           | Description                    |
| ---------------------------------------------------- | ------------------------------ |
| **expiration\_date**   String _ISO 8601 date in UTC_ | Expiration date of the coupon. |
